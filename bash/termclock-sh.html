termclock.sh - draws an analog clock from letters

When I started to learn Linux I met two commands : xclock and bc. And I got the idea to imitate the first using the second.

You will see nothing from the original bc based implementations. I am not really proud of those. The current version uses an “angle” table from which derives the needed sine & cosine values then precalculates the positions. This makes it faster and uses less resources. And of course, reduces the dependencies to tput only.

termclock.sh just indicates the time and indicates the time and… You have to press Ctrl-C to stop it.

Configuration

None.

Not even command line parameters exist.
